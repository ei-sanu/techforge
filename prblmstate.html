Problem Statements

Track 1: AI-Driven Fake News Detection

Challenge Background:

  • Misinformation spreads 6x faster than factual information
  • Current detection methods often lag behind viral spread
  • Source attribution remains difficult

Expected Deliverables:

  • NLP model to detect fake news with >85% accuracy
  • Network analysis algorithm to trace propagation patterns
  • Real-time monitoring dashboard prototype
  • Source attribution mechanism

Track 2: AI-based Adaptive Learning Platform

Challenge Background:

  • Rural areas face significant educational disparities
  • One-size-fits-all approaches fail to address diverse learning needs
  • Limited technology access requires innovative solutions

Expected Deliverables:

  • ML model that adapts to student learning patterns
  • Emotion recognition system for engagement monitoring
  • Offline-capable learning modules
  • Data analytics dashboard for teachers

Track 3: AI in EV Two-Wheeler

Challenge Background:

  • EV adoption faces range anxiety and performance concerns
  • Battery management remains a critical challenge
  • Rider experience needs enhancement

Expected Deliverables:

  • AI-powered battery optimization system
  • Smart route planning for energy efficiency
  • Predictive maintenance solution
  • Intelligent rider assistance features

Track 4: EV Two-Wheeler Rapid Production
